# SocialVibe - Premium Social Network Demo

## Overview
SocialVibe is a high-end social media feed demonstration featuring glassmorphism design, smooth animations, and local persistence.

## Tech Stack
- React + TypeScript + Vite
- Zustand (State Management + Persistence)
- Framer Motion (Animations)
- Lucide React (Icons)
- Vanilla CSS (Premium styling)

## Architecture
- `src/store.ts`: Centralized state for posts, likes, and comments.
- `src/data/feed.json`: Initial source of truth with Unsplash images.
- `src/App.tsx`: Main UI components with Framer Motion.
